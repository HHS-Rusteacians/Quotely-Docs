Welcome to Quotely's documentation!
===================================

**Quotely** (/Qu ot'eli/) is a Command line interface written in Java for ship manufactures to calculate
the price of how much a ship costs
It can be easily installed and work on most operating system platforms
and offers a *simple* and *intuitive* interface that anyone can use.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
